# Strategic District Field Guide

A mobile-friendly Streamlit app for conference prep and district leader conversations.

The app allows users to upload a district research Excel workbook and generate searchable district conversation cards.

Each district card includes:

- Lead-with conversation angle
- Strategic signals
- PCG / Emerald alignment
- Key contacts
- NEPQ-style discovery questions
- Listen-for cues
- Avoid cues
- Word export

## Expected Workbook Sheets

The app is designed to work with Excel workbooks that include:

- Strategic Indicators
- Master Scorecard
- District Contacts
- Basic District Info

Optional but helpful:

- CSI
- TSI
- Leadership and Governance

pip install -r requirements.txt
